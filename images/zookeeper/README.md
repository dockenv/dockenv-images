# Zookeeper

> Apache ZooKeeper is an open-source server which enables highly reliable distributed coordination.

## Usage

`$ docker run --name zookeeper --restart always -d -v $(pwd)/zoo.cfg:/conf/zoo.cfg ghcr.io/dockenv/zookeeper:latest`
